# 🎯 Quote Generator

A simple random quote generator built with **HTML**, **CSS**, and **JavaScript**.  
Fetches random quotes from [Advice Slip API](https://api.adviceslip.com/advice).

## ⚙️ Technologies
- HTML
- CSS
- JavaScript (fetch API)

## 🧠 Learning Goals
- DOM manipulation
- Async requests (`fetch`)
- Event handling
